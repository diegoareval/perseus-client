[Project Preview](https://perseus-client.vercel.app/)

** to sort elements click on the header of the table
## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run dev

# production
$ yarn run build
$ yarn run preview

# run  lint
$ yarn run lint

```
